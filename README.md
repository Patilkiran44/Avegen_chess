
# react-native-chess-Avegen

### Installation

Clone this repo

$ git clone https://github.com/Patilkiran44/Avegen_chess.git
$ cd Avegen_chess
$ npm install or yarn install
 
 if needed do pod install
 $ cd ios
 $ pod install

### How to start

$ npx react-native run-android
$ npx react-native run-ios
